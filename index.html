jQuery Tiny Notifier

I wanted a simple, really very simple Notifier, was lazy to find one, so built one

the CSS

.notifier {
    position: fixed;
    bottom: 0;
    right: 0;
    padding: 10px;
    text-align: justify;
    border: 1px solid #ddd;
    min-height: 30px;
    margin-bottom: 10px;
    margin-right: 10px;
    width: 400px;
    z-index: 999;
    cursor: default;
}
.notifier-success { background-color: #2A8893 !important; }
.notifier-error { background-color: #CF000F !important; color: #fff !important; }
.notifier-info { background-color: #3498DB !important; }

the jQuery 
jQuery(document).ready(function () {
    jQuery.fn.notify = function(notificationText, notificationType='success') {
        var notificationClassName = 'notifier notifier-'+notificationType;
        var div = '
'+ notificationText +'
';
        jQuery('body').append(div);
    }

    setTimeout(function () {
        $('.notifier').fadeOut('slow');
    }, 5000);
    // $('#notify-main').click(function() { $(this).fadeOut('fast') });
    hideThis = function ($this)
    {
        $($this).fadeOut('fast');
    }
});

and call like this

var test = 'testing jquery notification';
$('body').notify(test, '');
